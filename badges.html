Certification Badges

Trust signals that communicate product quality, sustainability certifications, and compliance standards.

Usage & Purpose

When to Use

Display quality indicators like MSC Certified, Organic, ASC Approved, or Sustainable. These are informational trust signals, not actionable status indicators.

Visual Style

Soft filled backgrounds (Success-200), no borders, very rounded pill shape (16px). Calm and reassuring appearance that doesn't compete with product cards.

Icon Placeholder

Rectangle icon area provided for logo/icon placement. Replace with actual certification logo or icon in your implementation.

Colors: Background uses Success-200 (#A8E6C5) for optimal contrast. Text uses Success-700 (#2C7048) for readability. This is NOT the same as status indicators which use bordered styles.

Accessibility Notes

  • • Color Contrast: Success-200 background with Success-700 text meets WCAG AA standards (contrast ratio 4.8:1).
  • • Icon Alt Text: If using images in the icon placeholder, always provide alt text describing the certification.
  • • Screen Readers: The badge text is automatically read. No additional ARIA labels needed for simple certification badges.
