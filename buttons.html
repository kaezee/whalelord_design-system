Buttons

Action elements that guide buyers through discovery, quoting, and ordering flows.

Three sizes optimized for desktop and mobile marketplace interfaces.

Primary

Main call-to-action for critical marketplace actions: Request Quote, Place Order, Confirm Purchase. The Waves blue builds buyer confidence.

<button class="btn btn-md btn-primary">Request Quote</button>
<button class="btn btn-md btn-primary" disabled>Disabled</button>
<button className="bg-waves-500 hover:bg-waves-700 text-white px-6 h-10 rounded-lg font-medium transition-all hover:underline">Request Quote</button>

Primary Outline

Secondary CTAs with Waves emphasis: Compare Products, View Details, Save to List. Hovers to Shore Ice for clear feedback.

<button class="btn btn-md btn-primary-outline">View Details</button>
<button class="btn btn-md btn-primary-outline" disabled>Disabled</button>
<button className="border border-waves-500 hover:bg-shore-ice hover:border-waves-700 text-pearl-800 px-6 h-10 rounded-lg font-medium transition-all hover:underline">View Details</button>

Secondary Outline

Tertiary actions with minimal emphasis: Cancel, Go Back, Skip Step. Neutral gray maintains focus on primary flows.

<button class="btn btn-md btn-secondary-outline">Cancel</button>
<button class="btn btn-md btn-secondary-outline" disabled>Disabled</button>
<button className="border border-border-neutral hover:bg-shore-base hover:border-border-hover text-pearl-800 px-6 h-10 rounded-lg font-medium transition-all hover:underline">Cancel</button>

Neutral

Minimal emphasis for supporting actions: Learn More, View FAQ, Download Spec Sheet.

<button class="btn btn-md btn-neutral">Learn More</button>
<button class="btn btn-md btn-neutral" disabled>Disabled</button>
<button className="bg-shore-base border border-border-default hover:bg-border-default text-pearl-800 px-6 h-10 rounded-lg font-medium transition-all">Learn More</button>

Fluid / Responsive Buttons

Full-width buttons for mobile checkout flows, form submissions, and stacked action layouts. Add btn-fluid class to any button variant.
