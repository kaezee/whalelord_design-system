Buttons

Primary interaction elements with complete state specifications. Available in multiple sizes and variants optimized for Tailwind + React.

Button Sizes

Buttons come in five sizes: xsmall (28px), small (32px), medium (40px), large (48px), and xlarge (56px).

HTML 
<!-- Xsmall (28px) -->
<button class="inline-flex items-center justify-center px-3 py-1.5 text-xs font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700">
  Xsmall
</button>

<!-- Small (32px) -->
<button class="inline-flex items-center justify-center px-4 py-2 text-sm font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700">
  Small
</button>

<!-- Medium (40px) - Default -->
<button class="inline-flex items-center justify-center px-6 py-2.5 text-base font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700">
  Medium
</button>

<!-- Large (48px) -->
<button class="inline-flex items-center justify-center px-8 py-3 text-lg font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700">
  Large
</button>

<!-- Xlarge (56px) -->
<button class="inline-flex items-center justify-center px-10 py-3.5 text-xl font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700">
  Xlarge
</button>
React + Tailwind 
const Button = ({ size = 'md', children, ...props }) => {
  const sizes = {
    xs: 'px-3 py-1.5 text-xs',
    sm: 'px-4 py-2 text-sm',
    md: 'px-6 py-2.5 text-base',
    lg: 'px-8 py-3 text-lg',
    xl: 'px-10 py-3.5 text-xl',
  };

  return (
    <button
      className={`inline-flex items-center justify-center ${sizes[size]} font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700 transition-colors`}
      {...props}
    >
      {children}
    </button>
  );
};

// Usage
<Button size="xs">Xsmall</Button>
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>
<Button size="xl">Xlarge</Button>

Button Variants

Primary buttons for main actions, secondary for alternative actions, and outline for tertiary actions.

HTML 
<!-- Primary -->
<button class="inline-flex items-center justify-center px-6 py-2.5 text-base font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700 transition-colors">
  Primary
</button>

<!-- Secondary -->
<button class="inline-flex items-center justify-center px-6 py-2.5 text-base font-medium text-pearl-900 bg-pearl-100 rounded-md hover:bg-pearl-200 transition-colors">
  Secondary
</button>

<!-- Outline -->
<button class="inline-flex items-center justify-center px-6 py-2.5 text-base font-medium text-pearl-900 bg-transparent border border-pearl-300 rounded-md hover:bg-pearl-50 transition-colors">
  Outline
</button>

<!-- Ghost -->
<button class="inline-flex items-center justify-center px-6 py-2.5 text-base font-medium text-kelp-500 bg-transparent rounded-md hover:bg-kelp-50 transition-colors">
  Ghost
</button>
React + Tailwind 
const Button = ({ variant = 'primary', children, ...props }) => {
  const variants = {
    primary: 'text-white bg-kelp-500 hover:bg-kelp-700',
    secondary: 'text-pearl-900 bg-pearl-100 hover:bg-pearl-200',
    outline: 'text-pearl-900 bg-transparent border border-pearl-300 hover:bg-pearl-50',
    ghost: 'text-kelp-500 bg-transparent hover:bg-kelp-50',
  };

  return (
    <button
      className={`inline-flex items-center justify-center px-6 py-2.5 text-base font-medium rounded-md transition-colors ${variants[variant]}`}
      {...props}
    >
      {children}
    </button>
  );
};

// Usage
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="outline">Outline</Button>
<Button variant="ghost">Ghost</Button>

Buttons with Icons

Buttons can include icons on either side. Icons should be 16×16px and use consistent stroke weight.

Icon Left

Icon Right

HTML 
<!-- Icon Left -->
<button class="inline-flex items-center justify-center gap-2 px-6 py-2.5 text-base font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700 transition-colors">
  <svg class="w-4 h-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
    <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M13 7l5 5m0 0l-5 5m5-5H6"></path>
  </svg>
  Request Quote
</button>

<!-- Icon Right -->
<button class="inline-flex items-center justify-center gap-2 px-6 py-2.5 text-base font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700 transition-colors">
  Continue
  <svg class="w-4 h-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
    <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M13 7l5 5m0 0l-5 5m5-5H6"></path>
  </svg>
</button>
React + Tailwind 
import { ArrowRight, Plus, Eye, Download } from 'lucide-react';

const IconButton = ({ icon: Icon, iconPosition = 'left', children, ...props }) => (
  <button
    className="inline-flex items-center justify-center gap-2 px-6 py-2.5 text-base font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700 transition-colors"
    {...props}
  >
    {iconPosition === 'left' && Icon && <Icon className="w-4 h-4" />}
    {children}
    {iconPosition === 'right' && Icon && <Icon className="w-4 h-4" />}
  </button>
);

// Usage
<IconButton icon={ArrowRight} iconPosition="left">
  Request Quote
</IconButton>

<IconButton icon={ArrowRight} iconPosition="right">
  Continue
</IconButton>

Loading States

Show loading spinners during asynchronous operations.

HTML 
<button disabled class="inline-flex items-center justify-center gap-2 px-6 py-2.5 text-base font-medium text-white bg-kelp-500 rounded-md">
  <svg class="animate-spin w-4 h-4" fill="none" viewBox="0 0 24 24">
    <circle class="opacity-25" cx="12" cy="12" r="10" stroke="currentColor" stroke-width="4"></circle>
    <path class="opacity-75" fill="currentColor" d="M4 12a8 8 0 018-8V0C5.373 0 0 5.373 0 12h4zm2 5.291A7.962 7.962 0 014 12H0c0 3.042 1.135 5.824 3 7.938l3-2.647z"></path>
  </svg>
  Loading...
</button>
React + Tailwind 
const Button = ({ loading, children, ...props }) => (
  <button
    disabled={loading}
    className="inline-flex items-center justify-center gap-2 px-6 py-2.5 text-base font-medium text-white bg-kelp-500 rounded-md hover:bg-kelp-700 disabled:opacity-70 transition-colors"
    {...props}
  >
    {loading && (
      <svg className="animate-spin w-4 h-4" fill="none" viewBox="0 0 24 24">
        <circle className="opacity-25" cx="12" cy="12" r="10" stroke="currentColor" strokeWidth="4"></circle>
        <path className="opacity-75" fill="currentColor" d="M4 12a8 8 0 018-8V0C5.373 0 0 5.373 0 12h4zm2 5.291A7.962 7.962 0 014 12H0c0 3.042 1.135 5.824 3 7.938l3-2.647z"></path>
      </svg>
    )}
    {children}
  </button>
);

// Usage
<Button loading={isSubmitting}>
  {isSubmitting ? 'Submitting...' : 'Submit'}
</Button>

Buttons on Dark Background

Light variants optimized for use on dark blue (#025E96) backgrounds.

HTML 
<!-- Light Fill -->
<button class="inline-flex items-center justify-center px-6 py-2.5 text-base font-medium text-pearl-900 bg-kelp-50 rounded-md hover:bg-kelp-100 transition-colors">
  Light Fill
</button>

<!-- Light Outline -->
<button class="inline-flex items-center justify-center px-6 py-2.5 text-base font-medium text-kelp-50 bg-transparent border border-kelp-50 rounded-md hover:bg-kelp-900/20 transition-colors">
  Light Outline
</button>
React + Tailwind 
// For use on dark backgrounds
const LightButton = ({ variant = 'fill', children, ...props }) => {
  const variants = {
    fill: 'text-pearl-900 bg-kelp-50 hover:bg-kelp-100',
    outline: 'text-kelp-50 bg-transparent border border-kelp-50 hover:bg-kelp-900/20',
  };

  return (
    <button
      className={`inline-flex items-center justify-center px-6 py-2.5 text-base font-medium rounded-md transition-colors ${variants[variant]}`}
      {...props}
    >
      {children}
    </button>
  );
};

// Usage
<LightButton variant="fill">Light Fill</LightButton>
<LightButton variant="outline">Light Outline</LightButton>
