Chips

Compact elements for representing tags, filters, or removable selections. Often used in search filters and multi-select inputs.

Basic Chips

Standard chips for displaying tags or selections.

Fresh Frozen Wild-Caught Sustainable
React + Tailwind 
// Basic Chip Component
const Chip = ({ children, variant = 'default' }) => {
  const variants = {
    default: 'bg-pearl-100 border-pearl-300 text-pearl-700',
    primary: 'bg-kelp-50 border-kelp-200 text-kelp-700',
  };

  return (
    <span 
      className={`inline-flex items-center gap-2 px-3 py-1.5 border rounded-full text-sm font-medium ${variants[variant]}`}
    >
      {children}
    </span>
  );
};

// Usage
<Chip>Fresh</Chip>
<Chip>Frozen</Chip>
<Chip variant="primary">Sustainable</Chip>

Removable Chips

Chips with close buttons for user-removable selections (e.g., applied filters).

Price: $50-$100 Location: Norway MSC Certified
React + Tailwind 
import { X } from 'lucide-react';

// Removable Chip Component
const RemovableChip = ({ 
  children, 
  onRemove, 
  variant = 'default' 
}) => {
  const variants = {
    default: 'bg-pearl-100 border-pearl-300 text-pearl-700',
    primary: 'bg-kelp-50 border-kelp-200 text-kelp-700',
  };

  const iconColors = {
    default: 'text-pearl-600 hover:text-pearl-900',
    primary: 'text-kelp-600 hover:text-kelp-900',
  };

  return (
    <span 
      className={`inline-flex items-center gap-2 px-3 py-1.5 border rounded-full text-sm font-medium ${variants[variant]}`}
    >
      {children}
      <button
        onClick={onRemove}
        className={`flex items-center justify-center w-4 h-4 ${iconColors[variant]} transition-colors`}
        aria-label="Remove"
      >
        <X className="w-3 h-3" />
      </button>
    </span>
  );
};

// Usage with state management
const FilterChips = () => {
  const [filters, setFilters] = useState([
    { id: 1, label: 'Price: $50-$100' },
    { id: 2, label: 'Location: Norway' },
    { id: 3, label: 'MSC Certified', primary: true },
  ]);

  const removeFilter = (id) => {
    setFilters(filters.filter(f => f.id !== id));
  };

  return (
    <div className="flex flex-wrap gap-2">
      {filters.map((filter) => (
        <RemovableChip
          key={filter.id}
          variant={filter.primary ? 'primary' : 'default'}
          onRemove={() => removeFilter(filter.id)}
        >
          {filter.label}
        </RemovableChip>
      ))}
    </div>
  );
};

Chips with Icons

Chips can include leading icons for enhanced visual identification.

North Atlantic John's Fishing Co. Verified Supplier
React + Tailwind 
import { MapPin, User, CheckCircle } from 'lucide-react';

// Chip with Icon Component
const IconChip = ({ 
  icon: Icon, 
  children, 
  variant = 'default' 
}) => {
  const variants = {
    default: 'bg-pearl-100 border-pearl-300 text-pearl-700',
    success: 'bg-coral-50 border-coral-200 text-coral-700',
    primary: 'bg-kelp-50 border-kelp-200 text-kelp-700',
  };

  return (
    <span 
      className={`inline-flex items-center gap-2 px-3 py-1.5 border rounded-full text-sm font-medium ${variants[variant]}`}
    >
      {Icon && <Icon className="w-4 h-4" />}
      {children}
    </span>
  );
};

// Usage
<IconChip icon={MapPin}>North Atlantic</IconChip>
<IconChip icon={User}>John's Fishing Co.</IconChip>
<IconChip icon={CheckCircle} variant="success">
  Verified Supplier
</IconChip>

Interactive Filter Chips

Clickable chips that toggle selection state, commonly used for multi-select filters.

React + Tailwind 
import { Check } from 'lucide-react';

// Interactive Filter Chip
const FilterChip = ({ 
  label, 
  isSelected, 
  onToggle 
}) => {
  return (
    <button
      onClick={onToggle}
      className={`inline-flex items-center gap-2 px-3 py-1.5 border rounded-full text-sm font-medium transition-all ${
        isSelected
          ? 'bg-kelp-500 border-kelp-500 text-white hover:bg-kelp-600'
          : 'bg-white border-pearl-300 text-pearl-700 hover:border-kelp-500 hover:bg-kelp-50'
      }`}
    >
      {label}
      {isSelected && (
        <Check className="w-4 h-4" />
      )}
    </button>
  );
};

// Usage with state
const FilterGroup = () => {
  const [selected, setSelected] = useState(['fresh', 'wild-caught']);

  const toggleFilter = (value) => {
    setSelected(prev =>
      prev.includes(value)
        ? prev.filter(v => v !== value)
        : [...prev, value]
    );
  };

  const filters = [
    { value: 'fresh', label: 'Fresh' },
    { value: 'frozen', label: 'Frozen' },
    { value: 'wild-caught', label: 'Wild-Caught' },
    { value: 'farm-raised', label: 'Farm-Raised' },
  ];

  return (
    <div className="flex flex-wrap gap-2">
      {filters.map((filter) => (
        <FilterChip
          key={filter.value}
          label={filter.label}
          isSelected={selected.includes(filter.value)}
          onToggle={() => toggleFilter(filter.value)}
        />
      ))}
    </div>
  );
};

Usage Guidelines

✅ Do

  • Use chips for tags, filters, and removable selections
  • Keep chip labels short and scannable
  • Use remove button for user-deletable chips
  • Show selected state clearly in filter chips
  • Group related chips together
  • Use consistent sizing across chip groups

❌ Don't

  • Don't use chips for primary navigation
  • Don't make chips too tall or wide
  • Don't use more than 2-3 chip variants in one view
  • Don't hide the remove button on hover (always visible)
  • Don't use chips for long text content
  • Don't make static chips look interactive

Accessibility

✓

Keyboard Support: Interactive chips are keyboard accessible with Tab and Enter/Space

✓

ARIA Labels: Remove buttons have descriptive aria-label attributes

✓

Focus Indicators: Clear focus states for keyboard navigation

✓

Selection State: Selected chips are announced to screen readers
