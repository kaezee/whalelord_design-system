Do's and Don'ts

‚úÖ Do

  • Use Primary 500 (#025E96) as main brand color
  • Maintain high contrast for text (4.5:1 minimum)
  • Use semantic colors for their intended purpose
  • Test color combinations with accessibility tools
  • Use neutral grays for most UI elements

‚ùå Don't

  • Use light blue (#E6F2F8) on white backgrounds
  • Mix semantic colors (red + green together)
  • Use color as the only indicator of state
  • Override primary brand color with custom blues
  • Use pure black (#000000) for text

Implementation

Use CSS custom properties for consistent color application:

/* Primary Blues */
.btn-primary {
  background: var(--color-primary-500);
  color: #FFFFFF;
}

.btn-primary:hover {
  background: var(--color-primary-700);
}

/* Semantic Colors */
.alert-warning {
  background: var(--color-warning-50);
  color: var(--color-warning-900);
  border: 1px solid var(--color-warning-500);
}

/* Text Colors */
.text-body {
  color: var(--color-neutral-900);
}

.text-secondary {
  color: var(--color-neutral-600);
}
