Button State Transitions

Buttons can transition between states to provide user feedback. Common pattern: Add to Cart → Added to Cart.

Interactive Button States

JavaScript 
// Add to Cart Transition
button.addEventListener('click', function() {
  // Show loading
  this.classList.add('btn-loading');
  this.disabled = true;
  
  setTimeout(() => {
    // Show success
    this.classList.remove('btn-loading');
    this.classList.add('btn-success', 'btn-success-pulse');
    this.innerHTML = '✓Added to Cart';
    
    // Reset after 2 seconds
    setTimeout(() => {
      this.classList.remove('btn-success', 'btn-success-pulse');
      this.innerHTML = '+Add to Cart';
      this.disabled = false;
    }, 2000);
  }, 1000);
});
