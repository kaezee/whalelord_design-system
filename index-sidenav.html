Compass Design System

A comprehensive design system with ocean-inspired blues, clear typography hierarchy, and accessible components. Built for developers, designed for clarity.

Design Principles

Ocean-Inspired

Color palette rooted in ocean blues - building trust through calming, professional aesthetics.

Clear Hierarchy

Consistent spacing scale (4px base), clear typography system, and logical component sizing.

Accessible

WCAG AA compliant color contrast, 44px minimum touch targets, semantic HTML structure.

Purpose-Driven

Every color, size, and component serves a clear purpose in creating intuitive user experiences.
