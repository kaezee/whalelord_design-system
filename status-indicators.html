Status Indicators

Actionable status badges that communicate order states, inventory levels, and urgent alerts.

Usage & Purpose

When to Use

Order status (Delivered, Processing, Cancelled), inventory alerts (Low Stock, Out of Stock), actionable states requiring user attention.

Visual Style

1px borders, barely-tinted backgrounds, medium rounded (8px). Crisp for actionable information that demands attention.

vs Certification Badges

Status indicators have borders and are action-oriented. Certification badges are soft/filled and informational.

Colors: Success (green) for delivered, Warning (amber) for processing/low-stock, Error (red) for cancelled/out-of-stock. All use semantic-100 backgrounds with semantic-700 borders/text.
