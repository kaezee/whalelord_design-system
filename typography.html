Line Heights

Four line height options for different content types.

Tight (125%) - Headings & Titles

The Future of Design Systems

Best for headings where vertical space is premium

Normal (150%) - Body Text & UI

This is the standard line height for most UI elements and body text. It provides good readability while maintaining efficient use of space. Perfect for buttons, labels, cards, and general interface text.

Relaxed (175%) - Long-form Content

Use this more spacious line height for articles, blog posts, documentation, and any long-form reading content. The extra breathing room significantly improves readability for extended reading sessions and reduces eye strain.

Loose (200%) - Emphasis & Quotes

"Design is not just what it looks like and feels like. Design is how it works. The extra space in this line height creates emphasis and allows text to breathe."

Do's and Don'ts

‚úÖ Do

  • Use DM Sans for all content and storytelling
  • Use Inter exclusively for numbers and prices
  • Use Roboto Mono only for IDs and code
  • Maintain minimum 16px for body text
  • Use tight line-height (125%) for headings
  • Use relaxed line-height (175%) for articles

‚ùå Don't

  • Mix DM Sans and Inter in body paragraphs
  • Use Inter for headings or general text
  • Use Roboto Mono outside technical contexts
  • Go below 14px for UI text (12px minimum)
  • Use loose line-height for compact UI
  • Override font weights arbitrarily

Implementation

Use CSS custom properties for consistent typography:

/* Content (DM Sans) */
h1, h2, h3, h4, h5, h6, p, button, .content {
  font-family: var(--font-heading);
}

/* Data Values (Inter) */
.price, .metric, .amount, .percentage {
  font-family: var(--font-body);
  font-weight: var(--font-weight-semibold);
}

/* Technical (Roboto Mono) */
.transaction-id, code, pre, .identifier {
  font-family: var(--font-mono);
}

/* Size Examples */
.heading-xl {
  font-size: var(--font-size-5xl);
  line-height: var(--line-height-tight);
}

.body-text {
  font-size: var(--font-size-base);
  line-height: var(--line-height-normal);
}

.long-form {
  font-size: var(--font-size-lg);
  line-height: var(--line-height-relaxed);
}
